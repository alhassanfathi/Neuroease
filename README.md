# 🧠 NeuroEase Health Monitor

> ⚠️ **Educational project only — not a real medical diagnostic tool.**

**NeuroEase Health Monitor** is a Python application developed for learning purposes in **Health Informatics** and **Python programming**. The project demonstrates how health-related data can be read from CSV files, analysed using rule-based logic, and presented in a clear and accessible format.

The application is designed as a prototype component of the **NeuroEase** platform, which aims to support people with neurological conditions, neurodivergence, and chronic illnesses through accessible digital health tools.

---

# 📌 Features

✅ Reads health measurements from a CSV file.

✅ Analyses blood sugar readings using predefined clinical thresholds.

✅ Calculates summary statistics including:

* Number of readings
* Minimum value
* Maximum value
* Average value
* Value range

✅ Displays a clear health summary in the terminal.
